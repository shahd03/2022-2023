Opdracht 3 :

  • Waarom staat de tekst in de blokken horizontaal gecentreerd?

- Door de text align middle.

  • Ga na dat een property vertical-align:center geen effect heeft.
    Waarom eigenlijk niet?

- Omdat het werkt op een inline of inline block.

  • Ga na dat vertical-align:middle er niet voor zorgt dat de tekst verticaal gecentreerd wordt.
    Wat doet vertical-align dan wel?

- Het plaatst het middelpunt van de div's op 1 horizontale lijn.

  • Wat is de ‘simpele’ manier om een één enkele regel tekst verticaal te centreren?

- Met de CSS regel line-height: 100px; (Toepast op de first-child)

  • Welke truc wordt vaak gebruikt om meerdere regels verticaal te centreren?

- Display flex table cell.

  • In 'styles.css' staan twee verschillende CSS regels die de height instellen van een element met class="area tall".
    Waarom krijgt de regel voor class 'tall' voorrang?

- Omdat het beter is om (tall) te specificeren.
