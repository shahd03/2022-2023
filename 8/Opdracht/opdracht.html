Opdracht :

**Mijn oplossing zit in de console.**


  • Welk formaat wordt er eigenlijk gebruikt om het Date object?

    • - Het formaat dat wordt gebruikt om het Date object in de JSON-string op te slaan is een ISO 8601 datumnotatie, namelijk "YYYY-MM-DD".


  • Wat gebeurt er als eenzelfde object meermaals voorkomt bij het opbouwen van de JSON tekst?

    • - Als eenzelfde object meerdere keren voorkomt bij het opbouwen van de JSON-string, dan wordt deze meerdere keren opgeslagen in de JSON tekst.
    Dit kan leiden tot een grotere bestandsgrootte van de JSON tekst en mogelijk tot onnodige redundantie. In het voorbeeld wordt het object "adres" bijvoorbeeld tweemaal opgeslagen in de JSON tekst.
